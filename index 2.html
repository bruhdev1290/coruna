Grid Layout Integer Overflow PoC

How to observe: This PoC triggers signed integer overflow (undefined behavior) in nsGridContainerFrame's track position accumulation. Effects vary by build type:
  • Debug build: Assertion failure (crash with stack trace)
  • ASan/UBSan build: Reports signed-integer-overflow
  • Release build: May show layout glitches or nothing visible (UB is silent)
Run with ./mach run --debug or build with --enable-address-sanitizer.
